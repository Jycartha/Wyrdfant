# Intention Field
Occasionally shortened to simply “**intention**”  

A classical field knitted to [the conscious plane](The_Aliffrume_Encyclopedia/Intuitive%20Phenomena%20&%20Theory/Pure%20Phlogistics/Conscious%20Plane.md), thought to be generated by each [fundamental phase](The_Aliffrume_Encyclopedia/Intuitive%20Phenomena%20&%20Theory/Pure%20Phlogistics/Fundamental%20Phase.md), which contributes to the curvature of the conscious plane at each point.
