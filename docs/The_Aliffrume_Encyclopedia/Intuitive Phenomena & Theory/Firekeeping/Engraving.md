# Engraving
The process by which a [firekeeper](Firekeeper.md) gradually increases the efficiency of an [engram](../Pure%20Phlogistics/Engram.md) through understanding and repeated usage as it "engraves" upon their soul plane, before eventually becoming second nature and entering the keeper's [domain](Engraving.md#Domain).

## Domain
The set of engrams which a firekeeper can implement without [flamescript](../Soulfire/Flamescript.md), often with greatly reduced cost and computation delay.
